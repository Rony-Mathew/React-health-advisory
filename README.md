## Project Name & Pitch

Health-Advisory

A training project used for creating a landing page regarding health advices.


## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Run Test Suite:  

`npm test`  

To Start Server:

`npm start`
